# B2-2: 位置参数与参数传递的语义模型

## B2-2.1: 什么是位置参数

在 bash 中, 位置参数是 Shell 在解析命令或脚本入口时自动建立的一组有序字符串.

当执行:

```bash
./script.sh a "b c" d
```

Shell在进入脚本前就已经完成了:

- 解析
- 展开
- 引号处理
- 单词拆分(在调用方)

随后, 把结果交给脚本进程,形成以下结构:

- `$0`: 脚本名(或函数名)
- `$1`: 第一个参数 -> `a`
- `$2`: 第二个参数 -> `b c`
- `$3`: 第三个参数 -> `d`
- `$#`: 参数个数 -> `3`
- `$@`/`$*`: 参数列表的整体表示

参数位置是 "调用点的结果", 不是脚本内部再解析一遍.
